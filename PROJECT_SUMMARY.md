## 🎯 游戏流程

```
开始游戏
    ↓
输入名字 → 获得初始皮卡丘
    ↓
主菜单
    ├─ 探索 → 遭遇野生宝可梦
    │   ├─ 捕捉 → 加入队伍
    │   ├─ 战斗 → 回合制对战
    │   └─ 逃跑 → 返回主菜单
    ├─ 查看队伍 → 显示宝可梦详情
    ├─ 查看背包 → 显示道具和金钱
    └─ 退出 → 游戏结束
```

## 🛠️ 开发指南

### 编译
```bash
cargo build                    # 开发版本
cargo build --release         # 发布版本 (优化)
```

### 运行
```bash
cargo run                      # 直接运行
./target/release/poke         # 运行编译后的二进制
```

### 测试
```bash
cargo test                     # 运行所有测试
cargo test --test integration_test  # 运行集成测试
```
